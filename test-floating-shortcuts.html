Floating Shortcuts Button Test

This page demonstrates the floating keyboard shortcuts button that appears in the bottom-right corner of the application.

How it works: Click the floating button (keyboard icon) in the bottom-right corner to see all available keyboard shortcuts. The button is always visible and accessible from any page in the application.

Current Active Shortcuts

  • Alt + N: New Quotation (opens modal)
  • Alt + P: New Product (opens modal)
  • Alt + C: New Customer (opens modal)
  • Alt + S: Settings
  • Alt + D: Dashboard
  • Alt + L: Product List
  • Alt + Q: Quotation List
  • Alt + U: Customer List

Benefits of Floating Button

  • Always Accessible: Available from any page without navigating to settings
  • Clean Settings Page: Removes duplicate shortcuts from settings
  • Better UX: Users can quickly check shortcuts without losing their current work
  • Visual Cue: The keyboard icon clearly indicates its purpose

Implementation Details

  • Button is positioned fixed in bottom-right corner
  • Uses z-index to stay above other content
  • Hover effects for better interactivity
  • Modal shows shortcuts organized by category
  • Includes helpful tips and usage information
